Here's how to embed a responsive iframe in Antora, by hard coding HTML and CSS in a pass through block. +
 +
// Source for this technique: https://benmarshall.me/responsive-iframes/
// https://neveruntilnow.com/antora/jamstack/latest/multimedia/multimedia-example.html
// https://neveruntilnow.com/kit/index.html
// https://neveruntilnow.com/docsite/markup/index.html
++++
<!-- HTML to embed an iframe -->
<div class="iframe-container">
  <iframe src="https://www.usps.com/" frameborder="0"></iframe>
</div>

<!-- CSS to make the iframe responsive -->
<style>
.iframe-container {
overflow: hidden;
padding-top: 100%;
position: relative;
}
.iframe-container iframe {
 border: 0;
 height: 100%;
 left: 0;
 position: absolute;
 top: 0;
 width: 100%;
}
</style>
++++

== Try it yourself

Copy this code into an AsciiDoc file. Then insert your choice of URL. The source for this technique is https://benmarshall.me/responsive-iframes/.

NOTE: Some websites do not allow their pages to be displayed inside an iframe. If you see a message saying that the connection was refused, you cannot use that address in an iframe. The following URLs should work for demo purposes:
+
* https://neveruntilnow.com/antora/jamstack/latest/multimedia/multimedia-example.html
* https://www.usps.com/ +
